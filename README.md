# Interview Scheduler

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```
npm start
```

## Scheduler Api must be installed as all data for the api is stored there.

## Dependencies
 -axios
 -classNames
 -Normalize.css
 -React
 -React-dom
 -react-Scripts
